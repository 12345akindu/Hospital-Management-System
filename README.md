# Hospital-Management-System
This Hospital Management System was created for Sri Lanka Government Hospitals using C# in Visual Studio and MS SQL Server. It helps hospital staff manage patient records, doctor details, and appointments in an easy and organized way. The system does not include billing and focuses on improving daily hospital operations.
